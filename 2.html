CSS

Cascading Style Sheets (CSS)is a simple mechanism for adding style (e.g., fonts, colors, spacing) to Web documents.These pages contain information on how to learn and use CSS and on available software. They also contain news from the CSS working group.

Starting with HTML + CSS teaches how to create a first style sheet. For a quick intro, try chapter 2 of Lie & Bos or Dave Raggett's intro to CSS. Or see examples of styling XML and CSS tips & tricks. W3C's Core Styles are simple style sheets ready for use. Web Standards Curriculum is a self-study course.
